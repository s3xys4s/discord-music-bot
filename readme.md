## Simple music bot for discord

#### 1. Install dependencies with a package manager of your choice
```
npm install
# or 
yarn install
```

#### 2. Create config.json
```
{
  "token": "<bot_token>"
}
```

Token can be obtained here: https://discord.com/developers/applications.

### 3. Start it
```
npm start
# or
yarn start
```
